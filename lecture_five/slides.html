Coding a one column layout.

<div id="main-column">
...
</div>

#main-column {
    width: 1000px;
    margin-left: auto;
    margin-right: auto;
}

This will be centered in the middle of the page. But sometimes, you'll want it to be left, or perhaps even right aligned. For that, we'll use floats.

Multi column layouts step 1: structure your content with <div>'s

Logically seperate your HTML content into separate columns by wrapping them in <div> tags, and assigning either an appropriate ID or Class to style them with CSS.

Then use float to position them! Huh?

Layouts with float.

  • Normal flow of an HTML document.
  • Only applies to block elements.
  • Top to bottom in the same order as they appear in HTML.

Layouts with float.

  • Floating removes an element from the normal flow.
  • Moves it as far to the left or the right as possible.
  • Remains contained inside any containing block.
  • Possible values: 
    float: left;
float: right;
float: none;

Layouts with float.

  • Elements flow around floated elements - think of text flowing around an inset image.
  • An element after a floated element in the normal flow moves up to occupy the space.
  • This isn't optimal for nice grid-based layouts.

Layouts with float.

  • Enter the clear property!
  • Clearing floated elements is a reset button for the normal flow.
  • Can work in either direction or both directions.
  • Possible values: 
    clear: left;
clear: right;
clear: both;

Layouts with float.

  • Floated columns cannot occupy more horizontal space than the width of their containing block.
  • Remember the box model!
  • The space an element occupies is equal to:
  • width + padding-left + padding-right + border-left + border-right + (maybe margin-left and margin-right depending on context)
  • As a general rule, keep your boxes simple and layout will be much easier.

Basic two column layout.

<div id="wrapper">
    <div id="right">...</div>
    <div id="left">...</div>
    <div class="clear"></div>
</div>
.clear { clear: both; }

#wrapper {
    width: 1000px;
    margin-left: auto;
    margin-right: auto;s
}

#right {
    width: 800px;
    float: right;
}

#left {
    width: 150px;
    float: left;
}

Basic three column layout.

<div id="wrapper">
    <div id="right">...</div>
    <div id="leftest">...</div>
    <div id="left">...</div>
    <div class="clear"></div>
</div>
#right {
    width: 650px;
    float: right;
}

#left {
    width: 150px;
    float: left;
}

#leftest {
    width: 150px;
    float: left;
    margin-right: 10px;
}

Absolute Positioning.

Absolute positioning allows you to position an element wherever you want, relative to it's parent container. Two rules:

  1. The parent container must be set to position: relative; or position: absolute;
  2. The element you are positioning must have position: absolute; and values for either top, left, right, or bottom

Absolute postioning

<div id="container">
    <div id="callout">...</div>
</div>
#container { position: relative; }
#callout {
    position: absolute;
    top: 300px;
    left: 100px;
}

Absolute postioning

You can go outside of a parent container by using negative values, as long as the parent container is set to overflow: visible.

<div id="container">
    <div id="callout">...</div>
</div>
#container { 
    position: relative;
    overflow: visible;
}

#callout {
    position: absolute;
    top: -50px;
    left: -50px;
}
