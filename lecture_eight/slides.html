What is JS?

  1. Another programming language.
  2. An aid in interacting with web browsers.

JS - Another programming language.

  • Store data (letters, numbers, etc).
  • Manipulate the data stored with (basic math, etc).
  • Save a sequence of operations to be executed at will (functions!).
  • A lot of other stuff you don't really need to know for this class.

JS - The web's interactive language.

  • Built into the programming langauge are two key objects, window and document.
  • window is an object representing the browser "window."
  • document is an object representing all of the HTML elements in the browser window.
  • Each object has pre-defined functions attached to them that allow you to alter what's happening in the browser.
  • I would start teaching you these functions, but it's somewhat futile.

jQuery

Making your life easier

jQuery is...

  • A library that adds another object just like window and document, and it's called $.
  • It's also called jQuery.
  • jQuery works by using CSS selectors to target HTML elements.
  • After you've selected elements, jQuery has functions to play with them.

Adding a Javascript File

  • Uses a script tag to load external Javascript files.
  • <script src="jquery.js"></script>
  • For this class, your javascript should be encapsulated as follows:
  • $(function() {
   // your code goes here. 
});
  • Hiding an element with jQuery:
  • $(function() {
    $('#id-of-element').hide();
});
