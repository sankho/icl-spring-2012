HTML Classes and IDs

  • Classes and IDs are attributes and can be applied to any tags within the <body> (like src or href).
  • Classes are repeatable and tags can have multiple classes.
  • IDs are unique - they can only be applied once on a page and can only have one value.

Class and ID examples.

<p class="foo"></p>
<div class="foo bar"></div>
<span id="baz"></span>

CSS

  • Stands for "Cascading Stylesheets".
  • CSS is written in .css files.
  • CSS files are attached to HTML files to alter their appearance.
  • This is achieved with the <link> tag.

<link> tags.

<link rel="stylesheet" type="text/css" href="file.css">

CSS syntax.

CSS rules have two parts:

  • A selector is used to match tags and it is followed by...
  • A declaration block which is a set of rules surrounded by "{" and "}" to be applied to tags matching the selector.
p {
color: red;
font-family: Arial, sans-serif;
}

CSS selectors.

Selector Matches
p Matches every <p> tag on the page.
.foo Matches every tag on the page with class="foo".
div.foo Matches every <div> tag on the page with class="foo".
#bar Matches the only tag on the page with id="bar".

CSS properties.

  • A property is an effect that can be applied to a tag.
  • Examples include color and line-height.
  • They are followed by a value: property: value;.
  • Don't forget the : and ;.
  • A list of most of the CSS properties available.

Blocks vs. Inlines

  • It's important to understand that there are two main display types and each element has one inherent.
  • A block element fills the width of its parent and forces new lines below it.
  • An inline element only occupies the dimensions it needs and wraps like normal text.
  • Block elements.
    div, p, ul, table, h1, blockquote, etc.
    Inline elements.
    a, span, b, i, code, strong, etc.

The Box Model

  • The box model defines how block elements (<div>, <p>, <ul>, etc.) are sized and displayed.
  • It consists of 4 properties: width and/or height, padding, border, margin.
  • The properties are accounted for from the inside of the box to the outside.

Creating a horizontally centered column.

This only works on block elements like div! There are 2 requirements:

  1. A static width.
  2. Automatic margins on left and right.

Creating a horizontally centered column.

<div class="page">...</div>
.page {
width: 960px;
margin-left: auto;
margin-right: auto;
}
