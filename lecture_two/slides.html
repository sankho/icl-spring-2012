Lecture Two.

The holy trinity of web standards.

  • HTML - a simple, human-readable language used to describe content and its structure (not visual structure!).
  • CSS - a more complex language used to describe a set of presentational style rules for HTML pages.
  • JavaScript - a programming language used to inject behavior into HTML pages (such as "what happens when I click this link").
    Not related to Java. It's stupid and confusing, I know.

The summarized trinity.

It is critically important to understand the different roles!

  • HTML: content.
  • CSS: presentation.
  • JavaScript: behavior.

Formal intro to HTML.

HTML syntax has 5 core concepts.

  1. Opening tags: <p>.
  2. Closing tags: </p>.
    Some tags, like <img> do not need a closing tag! Most do.
  3. Attributes: class="foo" or href="http://google.com/".
  4. Content: <p>Hello world!</p>.
  5. Nesting: Tags can go inside tags, which can go inside tags, which can...

There are some less important pieces, too.

  • Comments. These are ignored by the browser.
    <!-- This is a comment. -->

Basic HTML structure.

Every HTML page you create must begin with this basic structure.

<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <title></title>
    </head>
    <body>
    </body>
</html>

DOCTYPE

Tells the browser "this is an HTML page".

Must be the first line of the page and look exactly like:

<!DOCTYPE html>
<html></html>

Indicates that everything inside the pair is HTML.

<body></body>

The body makes up the contents of the web page.

<head></head>

The contents of this tag are not rendered on the page.

<title></title>

The title of the page is displayed in the browser bar or the tab. Never on the page!

<meta charset="utf-8">

It's not important to understand this in-depth. What this means for you is that you can use any characters you want and includes every human language character.

However, sometimes you need character entities for HTML special characters.

Character entities.

Symbol Description Entity
" quotation mark &quot;
' apostrophe &apos;
& ampersand &amp;
< less than &lt;
> greater than &gt;
