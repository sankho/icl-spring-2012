Review, Resume, and CSS3.

HTML.

Organizes and defines our content.

HTML.

  • HTML Tags wrap around content.
  • Tags can be nested within tags.
  • Tags consist of a type and attributes. 
    <div id="main-column">
  • By default tags are either block or inline.
    • block - take up the entire width of the containing tag
    • inline - stay “in line” with other elements
  • Tag types organize our content's meaning.
  • Tag attributes of IDs or Classes organize our content according to our rules.

CSS.

Defines the visual appearance of our content.

CSS

  • uses a selector to target HTML tags using the tag type, class, or id attribtues
  • selectors can be joined and nested to represent relationships
  • after a selector statement, attributes are defined within {}'s
  • attributes have the following format : "key: value;" 
    div#example-id .example-class {
    color: #e1e1e1;
}

Columned Layouts.

Columned Layouts.

  • Separate content in HTML into columns logically by wrapping <div> tags around each desired column.
  • Keep column tags on the same nested level within a single containing tag.
  • Target the columns with CSS and use the float property to float columns as far to the left or right as possible of the container tag.
